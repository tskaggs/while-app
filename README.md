# While

Healthcare EMR/EHR connectivity platform — UI-only MVP built with Nuxt 4.

## Features

- **Overview dashboard** — usage KPIs, message throughput chart, connection health
- **Sandbox / Live switcher** — global environment toggle with mock data per environment
- **Connections** — partner integrations with tunnel status, topology map, flight checks
- **Request Support** — multi-step wizard to fast-track healthcare partner onboarding
- **Logs** — integration log stream with PHI anonymization toggle
- **Compliance** — BAA status, checklist, audit log
- **Developer Docs** — static API documentation via Nuxt Content

## Stack

- Nuxt 4.4.6
- Nuxt Content 3
- Unovis (charts)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Note

This is a **UI-only** phase — all data is mock/static. No backend API or authentication.
